- Real-time sentiment prediction (89% accuracy)
- REST API with Swagger docs
- Containerized with Docker
- CI/CD via GitHub Actions

## 🛠️ Quick Start

docker build -t sentiment-api . && docker run -p 5000:5000 sentiment-api
